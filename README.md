<h1>Movie Picker</h1>
Веб-приложение на Next.js и React для подбора случайного фильма на вечер. Пользователь может выбрать жанр (или оставить «Любой»), нажать кнопку и получить случайный фильм с описанием, рейтингами и метаданными. Если фильм не подошёл — можно запросить другой (в том же жанре) или вернуться на главную и выбрать другой жанр.
<p>ссылка на сайт: https://movie-picker-sigma.vercel.app/</p>
